Common Issues

Athena doesn't create Check Runs on my PR

Make sure the GitHub App has the pull_request event enabled under Permissions & Events. Also verify the app is installed on the repository (Settings → Integrations → GitHub Apps).

Check Run stays "in progress" forever

This usually means the pipeline encountered an error. Check your server logs for the specific error message. Common causes: invalid private key configuration, database connection issues, or missing permissions.

Workflow dispatch doesn't trigger

Ensure ENABLE_WORKFLOW_DISPATCH=true is set, the athena-dynamic-ci.yml workflow exists in your repo's main branch, and the app has Actions: Write permission.

Response Time

We aim to respond to GitHub Issues within 48 hours. Critical issues (service outages, security vulnerabilities) are prioritized.
